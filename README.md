# La Liga 2024/25 Analysis

## Overview
This project analyzes the final standings and player statistics from the La Liga 2024/25 season. Using Python and data visualization techniques, we explore team performances, player achievements, and key insights.

## Data Sources
- `la_liga_2024_25_final_standings.csv`: Contains final league standings, including team positions, matches played, wins, draws, losses, goals scored, and points.
- `la_liga_2024_25_player_stats.csv`: Includes individual player statistics such as top scorers, assist leaders, and clean sheet records.

## Dependencies
Ensure you have the following Python libraries installed:
```bash
pip install numpy pandas matplotlib seaborn
```

## Key Insights
1. **Top Teams**: Barcelona won the league with 89 points, followed closely by Real Madrid and Atlético Madrid.
2. **Top Scorer**: Kylian Mbappé led the league with 31 goals.
3. **Best Defense**: Barcelona conceded the fewest goals (30).
4. **Assist Leader**: Lamine Yamal topped the assist charts with 13 assists.
5. **Balanced Teams**: Mallorca and Villarreal displayed strong performances relative to their expected rankings.

## Visualizations
- Bar charts showing top goal scorers
- Pie chart of assist leaders per team
- Regression analysis on goals scored vs. final position
